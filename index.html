About Me

Yo, Halo! Saya Horasio Nissi Immanuel, mahasiswa Sains Data di Universitas Negeri Surabaya, dengan spesialisasi dalam analisis data, machine learning, dan pengembangan solusi berbasis AI. Saya tidak hanya menjelajahi dunia data, tetapi juga mengintegrasikannya dengan teknologi robotika melalui keanggotaan saya di Rengganis Robotic Club UNESA. Di sinilah saya mengasah kemampuan dalam merancang sistem cerdas, mengoptimalkan algoritma, dan menciptakan solusi berbasis otomatisasi.

Saya menguasai Python, R, SQL, serta alat visualisasi tingkat tinggi seperti Looker Studio dan Power BI untuk mengekstrak wawasan mendalam dari data. Di bidang robotika, saya berkontribusi dalam pengembangan sistem cerdas dan otomatisasi, termasuk penerapan machine learning untuk meningkatkan efisiensi dan presisi robot.

Portofolio saya mencakup proyek analisis prediktif, visualisasi data tingkat lanjut, serta algoritma machine learning dan robotika. Dengan fondasi yang kokoh dalam sains data dan teknologi, saya berkomitmen untuk menciptakan solusi inovatif dan revolusioner yang menghubungkan data dan kecerdasan buatan demi masa depan yang lebih cerdas dan efisien.
